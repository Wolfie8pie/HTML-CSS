Wat heb ik geleerd met keuzenvak HTML/CSS(reflectie)

Dit was de eerst keer dat ik met HTML/CSS ging werken. In het begin moest ik er aan wennen om met Visual Studio code en Github Desktop te werken.

Ik snapte eerst niet wat voor code ik moest gebruiken en ik had moeite met mijn design te veranderen in HTML/CSS.

Ik ben meer gewend om te werken met een grafisch programma zoals photoshop of illustrator dus werken met HTML/CSS was iets heel nieuwes en anders.

Ik vond het wel leuk om nieuwe programma’s en dingen te leren.

Door deze keuzevak heb ik de basics van HTML/CSS geleerd en ben blij dat ik nu wat ervaring heb.

Alles wat ik in deze lessen heb geleerd was nieuw voor mijn zoals, Maken van CSS, websites, foto’s en hyperlinks erin zetten.

Ik heb ook geleerd dat w3schools fijn is voor leren over HTML en CSS wanneer je iets wilt opzoeken.

Actiepunten

  • Gebruik w3schools als je iets wilt weten over HTML/CSS
  • Kijk goed naar je codes dat je niet iets vergeet zoals een "<",">",";"
  • Zorg dat je codes georganiseerd zijn zodat je niet door de waar raakt
  • Zorg dat je fotos in de zelfde folder staan met je index en css
