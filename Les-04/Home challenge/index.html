    Introductie

    Voor mijn HTML/CSS les 4 moest ik een probleem uitleggen en vertellen dat ik heb ervaart met HTML en CSS. Daarna heb ik ook tips geschreven die je ook kan helpen tijdens het uitvoeren van HTML/CSS.

    Mijn probleem

    Deze periode was de eerste keer dat ik HTML/CSS les had gekregen. De problemen dat ik had was dat wanneer ik een referentie met een hyperlink probeerde te maken was dat er soms een opsommingsteken aan het beginning van de referentie zat. Ik heb een stappenplan geschreven hoe je deze problem kan oplossen.

    Stappen

      • 1. Kijk eerste je index.html op je browser om na te kijken als er bij het begin van de referentie een opsommingsteken is.
      • 2. Rechter muis + options( voor mac) en klik op de pagina tot je de optie inspect ziet.
      • 3. Klik er op. Nu moet je html code zien op de rechter kant van je computer scherm.
      • 4. Klik waar je de referentie ziet zodat je weet welke stukje van je code verkeerd is.
      • 5. Ga terug naar je programma waar je jou html of css invoert. Voor mij was dat Visual Studio Code.
      • 6. Kijk naar deze voorbeeld: om te zien of er iets anders aan je code is.
      • 7. Als het niet is gelukt om het te veranderen kan je altijd de website w3schools gebruiken om je te helpen.

      Tip

      Tip: Vergeet niet dat je altijd aan je docent of de persoon die de les geeft kan vragen voor hulp.
